Il faut transférer le contenu d'un texte source html dans un texte cible markdown.

Dans le fichier source html ci dessous, identifie la balise <div itemprop="articleBody">. Elle contient l'information. Dans cette balise
- le contenu des balises <p>...</p> sont des consignes de déplacement, des informations ou des questions
- le contenu des balises <p><i>...</i></p> sont des réponses à des questions

L'objectif est de récupérer le texte des consignes et le texte des réponses dans le fichier source.

Dans le fichier cible :
- les consignes de déplacement doivent être insérées dans une balise <div class="go">
- les informations doivent être insérées dans une balise <div class="interest">
- les questions et les réponses associées doivent être formattées comme ci dessous

<details class="question">
<summary>Texte de la question</summary>
Texte de la réponse associée
</details>

Lorsque c'est terminé, extrait le titre du nom du fichier source pour fabriquer le nom du fichier cible

Par exemple :
- source : "Balade n°75 - Du métro Rambuteau à République - solution - Paris le nez en l'air"
- cible : 75-du-metro-rambuteau-a-republique.md

insère ensuite l'en tête, un titre et une balise <style>

---
title: Du métro Rambuteau à République
---

<style>
.go {
 border: 1px solid black;
 padding: 10px;
 text-align: center;
 background-color: honeydew;
 margin: 20px;
}
.question {
 border: 1px solid black;
 padding: 10px;
 text-align: center;
 background-color: lightcyan;
 margin: 20px;
}
.interest {
 border: 1px solid black;
 padding: 10px;
 text-align: center;
 background-color: papayawhip;
 margin: 20px;
}
</style>





Voilà le texte
